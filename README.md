##运行步骤

1.克隆GitHub仓库

```
git clone https://github.com/kiwenlau/kubernetes-shell
cd kubernetes-shell/
```

2.下载Docker镜像

```
sudo docker pull kiwenlau/kubernetes-shell:1.0.7
```

3.运行Kubernetes

```
chmod +x run.sh
./run.sh
```

4.测试kubernetes

```
chmod +x test-kubernetes.sh
./test-kubernetes.sh 
```
